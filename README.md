# Hotel_management_system
This project is a Hotel Reservation System developed using Java programming language with JDBC to connect MySQL as the backend database. The main goal of this project is to learn how Java applications interact with databases in a real-world environment. It is a console-based, menu-driven application that allows the user to manage hotel reservations in a simple and effective way.
